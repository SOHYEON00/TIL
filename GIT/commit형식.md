# git commit 형식

git commit 방식에 대해 알아보자.

여러가지 깃 커밋 룰이 있지만, 최근에 여러 회사의 기술 블로그를 보다가 알게 된 커밋 룰에 대해 써보고자 한다.
(앞으로 내 깃에 이렇게 입력하려고..)


그동안 커밋할 때마다 어떻게 커밋해야하나 하며 매번 나만의 룰이 항상 바뀌었다.
이는 당연히 공식적으로 커밋 룰을 정하지 않고 작성하지 않아서 이겠지..!
